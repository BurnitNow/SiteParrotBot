Parrot Bot

What is?

It's a discord bot that can give you the color palette of images, you can use the commands with the prefix "+"

What it does?

Gives you the color palette of an image
-Reply to a message containing a image with the command "+getcolor","+color" or "+colors" and if you want more colors just reply to the answer with "+more"
It can give you the name of an color
-Use the command "+name (hex)" example: "+name #FF0000"
